Mohon diperhatikan, jika menggunakan fitur ini. PDF anda menjadi tidak editable lagi karena sudah di-compress. Tutup slider ini atau tekan tombol Batal di bawah untuk membatalkan fitur.

Penjelasan

Website ini secara khusus digunakan untuk menggabungkan dua atau lebih file PDF. Kamu bisa drag and drop saja file pdfmu atau tekan field itu untuk mengupload file pdf kamu. Kamu juga bisa mengatur susunan PDFmu. Ada fitur untuk mengatur ukuran PDFmu juga di sini. Namun harus diperhatikan, output PDFmu tidak bisa diedit lagi jika file yang kamu ingin merge adalah PDF dari Adobe Illustrator. Karena secara teknis, PDFmu akan di-compress dengan fitur ini. Tutup saja fiturnya jika PDFmu ingin bisa diedit kembali.
